# ✅ Wordle - Clone Game

Wordle clone application built with **React + TypeScript**. Made with educational purposes!

This application allows users to:

- Enter letters creating a word
- Verify if the letters are correct
- Verify if any of the letters is in any of the cells
- Solve the puzzle or know the correct answer

---

## 🔗 Structure

The application follows a clear separation:

- **App**
  - Manages the game
  - Contains the logic
  - Connects the different components

- **Grid**
  - Handles user input
  - Lets the user see the letters
  - Reveal the words typed

- **KeyBoard**
  - Select letters by clicking
  - Know which letters are in the grid or in the right position
